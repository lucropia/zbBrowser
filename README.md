# zbBrower

A CEF3 Browser Wrap for window widget.

Dependence:
    
    1. cef3 build, you have libcef.lib, libcef_dll_warper.lib
    2. cef3 include files

Howto build:
    
    you have Visual Studio or clang windows. and GNU make execute file.
    see Makefiles
